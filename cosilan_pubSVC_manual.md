## pubSVC Manual

### 请求 URL
```
http://ip:port/pubSVC.php?Command&api_key={api_key}&Paramters
```

* Response
平台返回数据统一采用 JSON 格式数据。
```
{
	"code": "{response code}",
	"message": "{message}",
	"records": "{number of records}",
	"data":{data}
}
```

* response code  

|code|message|description|
|-----:|---|---|
|1|OK| 正确调用，并正常返回数据|
|1001|Invalid API Key|API Key 无效|
|1002|DB select Error|数据库 错误|
|1003|Invalid Query|Query文章 错误|
|1004|Invalid Parameter|因数 错误|


* Number of records : data 的 数量
```
Data : [{},{},……{}] 
```


#### getSquare 
```
http://ip:port/pubSVC.php?getSquare&api_key={api_key}
```
获取所有广场的信息，如parameter “code={square_code}”, 可以获取指定广场信息
Parameter : code = {square_code}
```
http://ip:port/pubSVC.php?getSquare&api_key={api_key}&code={code}
```


#### getStore
```
http://ip:port/pubSVC.php?getStore&api_key={api_key}
```
获取所有店铺的信息，如parameter “code={square_code}”, 可以获取指定店铺信息
Parameter : code = {store_code}

  
#### getCamera
```
http://ip:port/pubSVC.php?getCamera&apiey={api_key}
```
获取所有摄像机的信息，如parameter “code={camera_code}”, 可以获取指定摄像机信息
Parameter : code = {camera_code}

 
#### getCountingDB
```
http://ip:port/pubSVC.php?getCountingDB&apiey={api_key}&from={from}&to={to}
```
获取计数数据，parameter “from” 跟 “to” 是 必需的， 如parameter “camera_code={camera_code}”, 可以获取指定摄像机信息, 如counter_label=｛counter_label｝可以指定的 counter_label.
“counter_label” 是 “entrance”/ “exit” / “outside” / “none” 中 一个。
Parameter :
	* from: 开始日期或者 时间
	* to: 终止日期或者 时间
	* camera_code： 摄像机 编号
	* counter_label: 计数器 标签

数据是 每个 10分钟采样的数据， 如果 0数据(counter_val = 0) 不获取数据

  
#### getAgeDB
```
http://ip:port/pubSVC.php?getAgeDB&apiey={api_key}&from={from}&to={to}
```
获取年龄数据，parameter “from” 跟 “to” 是 必需的， 如parameter “camera_code={camera_code}”, 可以获取指定摄像机信息
Parameter :
	* from: 开始日期或者 时间
	* to: 终止日期或者 时间
	* camera_code： 摄像机 编号

数据 data 的 “age” 形式为 [a,b,c,d,…]， a 代表 0岁， b 代表 1岁… 总共 0~99岁
数据是 每个 10分钟采样的数据， 如果 0数据(age = [0,0,0,….0]) 不获取数据

 
#### getGenderDB
```
http://ip:port/pubSVC.php?getGenderDB&apiey={api_key}&from={from}&to={to}
```
获取所有性别的信息，如parameter “code={camera_code}”, 可以获取指定摄像机信息
Parameter :
	* from: 开始日期或者 时间
	* to: 终止日期或者 时间
	* camera_code： 摄像机 编号
数据是 每个 10分钟采样的数据， 如果 0数据(male=0, female=0) 不获取数据

 

#### getSnapshot
```
http://ip:port/pubSVC.php?getSnapshot&apiey={api_key}
```
获取所有摄像机抓拍图片，如parameter “code={camera_code}”, 可以获取指定摄像机信息
Parameter :
	* code = {camera_code}

图片形式为 :"data:image\/jpg;base64,~~~~ ", base64 encoded image string.

 

#### getHeatmap
```
http://ip:port/pubSVC.php?getHeatmap&apiey={api_key}&from={from}&to={to}
```
获取所有热力图数据，如parameter “code={camera_code}”, 可以获取指定摄像机信息

Parameter :
	* from: 开始日期或者 时间
	* to: 终止日期或者 时间
	* camera_code： 摄像机 编号

数据是 每个小时采样的数据
